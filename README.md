# vue3-shopmall-app

## 项目简介
```
本项目是移动端电商系统，包含首页、商品分类、购物车、个人中心等模块。
主要功能有：商品信息浏览、商品信息搜索、商品分类、购物车、商品结算、个人信息修改、登录等。
主要技术栈：vue3 + ts + vant + mockjs。
```
